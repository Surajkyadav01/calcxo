# 🚀 Calcxo – Smart Calculator Web App

Calcxo is a modern and user-friendly calculator web application designed to provide fast and accurate calculations with a clean interface. It is inspired by the functionality of the Google Calculator and aims to deliver a smooth and responsive user experience.

## ✨ Features

- 🧮 Basic Calculator (Addition, Subtraction, Multiplication, Division)
- 📊 Scientific Calculator Functions
- 💰 GST Calculator
- 🏷️ Discount Calculator
- ⚡ Fast & Responsive UI
- 📱 Mobile-Friendly Design

## 🎯 About the Project

Calcxo is built as a personal project to enhance web development skills and create a practical tool for everyday use. The goal was to replicate and extend the functionality of modern calculators like Google Calculator with additional useful features.
